Vidio

Vidio is a leading streaming services in Indonesia. I mainly focused on the recommendation system, though I also developed churn prediction and content performance prediction.

Notable Projects

  • Recommendation System: Designed and implemented a collaborative filtering recommendation system using cosine similarity, which improved user engagement metrics significantly.
  • Churn Prediction: Developed a machine learning model to predict user churn with 90% accuracy, enabling the marketing team to implement effective retention strategies.

Tools

Tools that are mainly used are.

  • Python for Data Analysis and Data Scientist, e.g. pandas, sk-learn, and tensorflow.
  • Deployment Tools, Flask, Dataproc Google Cloud Platform
  • Data Engineering Tools, PySpark, airflow
  • Visualization Tools, Redash, Tableau, Metadata
